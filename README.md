# 📊 Customer Segmentation using SQL & Power BI

## 📌 Objective

Segment customers based on demographic and behavioral data to understand different customer groups and support targeted business strategies.

---

## 🛠 Tools Used

* SQL (MySQL)
* Power BI
* Excel

---

## 📂 Dataset

* Customer dataset including Age, Gender, Profession, Spending Score, Family Size, and Work Experience

---

## 🧹 Data Cleaning

* Handled missing values using SQL (COALESCE)
* Replaced null values with average or default values
* Created a cleaned dataset for analysis

---

## 📊 Customer Segmentation

Customers were grouped using business rules based on:

* Age
* Spending Score
* Family Size
* Work Experience

### Segments Created:

* Young Low Spenders
* Affluent Professionals
* Family Customers
* Low Engagement Seniors
* Average Customers

## 🔍 Key Insights

* High spending customers form a smaller but valuable segment
* Family customers represent a significant portion of the dataset
* Older customers with low spending show lower engagement
* Different segments show different behavior patterns

---

## 💡 Recommendations

* Target high-value customers with premium offerings
* Improve engagement strategies for low-spending segments
* Focus marketing efforts based on customer segments

---

## 🚀 Project Outcome

This project demonstrates how customer data can be segmented using SQL and visualized using Power BI to generate actionable insights.
